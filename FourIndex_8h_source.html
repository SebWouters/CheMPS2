97  /*The following conventions are used for storage:
98  - 8-fold permutation symmetry: V_ijkl = V_jilk = V_ilkj = V_lijk = V_kjil = V_jkli = V_klij = V_lkji
99  - Reorder indices until V_ijkl (I_i x I_j = I_k x I_l --> stored per center symm block):
100  - I_i <= I_j <= I_l and I_i <= I_k
101  - Icenter == Itriv : I_i == I_j and I_k == I_l
102  - Icenter > Itriv : I_i < I_j and I_k != I_l
103  - Vmat[Icenter][I_i][I_k] --> only created if ordering of all sectors is ok (I_i <= Icent x I_i ; I_k >= I_i ; Icent x I_k >= I_cent x I_i)
104  - Once the order is established based on symmetry sectors, the order within symmetry sectors has to be set too:
105  - Case Icenter == Itrivial :
106  - If I_i == I_j == I_k == I_l : index (within symm block) i smallest; l>=j>=i and k>=i ; if i==j then l>=k
107  - Vmat[Itriv][I_i][I_i][i + k(k+1)/2]
108  - if i==j --> [j-i][l-k]
109  - if i< j --> [j-i][l-j]
110  - If I_i == I_j < I_k == I_l : index i<=j ; if i<j then k and l fixed ; if i==j then l>=k
111  - Vmat[Itriv][I_i][I_k][i + nOrbWithIrrepI_i * k]
112  - if i==j --> [j-i][l-k]
113  - if i< j --> [j-i][l]
114  - Case Icenter > Itrivial (I_i < I_j and I_k != I_l) :
115  - If I_i == I_k and hence I_j == I_l : index k>=i and index l>=j
116  - Vmat[Icent][I_i][I_i][i + k*(k+1)/2][j][l-j]
117  - If I_i < I_k and hence I_j < I_l : fixed by block order
118  - Vmat[Icent][I_i][I_k][i + nOrbWithIrrepI_i * k][j][l] */
